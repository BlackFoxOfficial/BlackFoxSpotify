# BlackFoxSpotify

BlackFox Spotify — это легковесный, кастомный desktop-клиент для Spotify Web Player без рекламы, написанный на C# (WinForms) с использованием движка Microsoft Edge WebView2.

# ✨ Ключевые возможности (Features)
```
Встроенный Ad-Blocker: Блокировка аудио-рекламы и трекеров аналитики на уровне перехвата сетевых запросов (WebResourceRequested).

Custom Borderless UI: Полностью кастомный интерфейс окна без стандартных рамок Windows, но с сохранением нативной тени (Drop Shadow) и возможности изменения размера (Resize).

Темная тема по умолчанию: Принудительное включение Dark Mode через аргументы запуска Chromium.

Легковесность: Отключены лишние функции браузера (сохранение паролей, статус-бар) для экономии ресурсов.
```
